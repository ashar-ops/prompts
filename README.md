# prompts

A curated collection of system prompts designed to make AI models more useful, honest, and effective for humans.

## What's in this repo?

Each prompt in the [`prompts/`](./prompts) folder is a ready-to-use system prompt you can paste into your AI assistant (ChatGPT, Claude, Gemini, etc.) to give it a specific persona or set of behavioral rules.

## Available Prompts

| File | Description |
|------|-------------|
| [the-thinking-partner-persona](./prompts/the-thinking-partner-persona) | A strict, radically honest, strategically-minded advisor persona. Enforces epistemic honesty, goal guardianship, intellectual backbone, and actionable execution over 21 non-negotiable behavioral rules. |

## How to Use

1. Open the prompt file you want.
2. Copy the full contents.
3. Paste it as the **system prompt** (or first user message) when starting a new conversation with your AI assistant.
4. The AI will adopt the persona and follow the rules defined in the prompt.

## Contributing

Have a prompt that makes AI genuinely more useful? Open a pull request and add it to the `prompts/` folder.
